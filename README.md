Projet : Prédiction des Cyberattaques (Apprentissage Automatique avec Python)

Objectif : Développer un modèle d'apprentissage automatique pour détecter et prédire des cyberattaques.

Technologies : Python, Jupyter Notebook, NumPy, pandas, scikit-learn, TensorFlow.

Méthodologie : Application d'une méthodologie d'apprentissage machine (préparation, modélisation, évaluation) utilisant un jeu de données Excel de 40 000 lignes pour prédire les attaques.

Résultat Attendu : Créer un modèle capable de classifier le trafic réseau comme normal ou attaque, en comprenant le cycle complet d'apprentissage machine : données → modèle → évaluation → amélioration.
